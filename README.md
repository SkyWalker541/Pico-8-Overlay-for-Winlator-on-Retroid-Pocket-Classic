# Pico-8-Overlay-for-Winlator-on-Retroid-Pocket-Classic
A Pico-8 Overlay for Winlator. Compatible with Retroid Pocket Classic, Retroid Pocket Mini, and Ayaneo Pocket DMG

Download this app for one on the three devices above. Apply the PNG, and adjust the settings to your likeing.
There is a setting, in the app, that will give you an ON/OFF toggle in the quick-menu for android.
